CS4483 Individual Prototype - Mountain Treasure

These instructions are for players playing on Windows 10
--------------------------------------------------------

To download the game: https://drive.google.com/file/d/1cyPZQLUKjK3O6YxU3S6JTYHmkoubU0pJ/view?usp=sharing

When you download it, unzip the folder onto your computer.
To play the game, go into the Mountain Treasure folder and run the application "Mountain Treasure"

For the controls, its the arrow keys to move and space bar to attack

The textures,tiles, and sound effects used for this game were available for free from the link below:
https://github.com/uheartbeast/youtube-tutorials/blob/master/Action%20RPG/Action%20RPG%20Resources.zip
The current textures could be changed later on as it does not fit the current mountain theme. This texture was used as a filler to help create the mechanics of the game and how it would feel.
